# Weather Forecast App

Built with React,NodeJs, Uses OpenWeatherMap's API and google/generative-ai for suggestions.


A modern weather forecasting application built with React that fetches real-time weather data from the OpenWeatherMap API and provides intelligent suggestions using the Google Generative AI API.

🚀 Features

🌍 Search weather by city name

📍 Get real-time weather forecast (temperature, humidity, wind speed, condition, etc.)

💡 AI-powered suggestions (e.g., what to wear, whether to carry an umbrella, activity ideas)

📱 Responsive UI for desktop and mobile


🛠️ Tech Stack

Frontend: React,CSS

APIs:

OpenWeatherMap API
 – for real-time weather data

Google Generative AI API
 – for intelligent activity suggestions

Others: Axios / Fetch for API calls

📦 Installation

Clone this repository:

Install dependencies: npm install

Add your created  WEATHER_API_KEY and APP_GOOGLE_API_KEY 

Run the app:npm start


🌤️ Usage

Enter a city name in the search bar.

View the weather forecast for that city.

Get AI-powered suggestions (e.g., "It's raining today, don’t forget an umbrella!").
